# web-api-example

## Run

```
dotnet watch run
```

## Database

This project using **in-memory database** for domonstration.

## API

### GET Todo

```
GET
/api/todo
/api/todo/{id}
```

### POST new todo

```
POST
/api/todo
```

![POST method](https://github.com/iamgoangle/dotnet-core-webapi-example/blob/master/web-api-example/README/Postman.png?raw=true)

### Update existing todo

```
PUT
/api/todo/{id}
```
